# Chrome Web Store 隐私权规范说明

## 单一用途说明

本扩展程序（X Data Scraper）的单一用途是：**帮助用户管理和备份自己的 Twitter/X 账户推文数据**。

具体功能包括：
- 从用户自己的 Twitter/X 账户分析页面提取推文、图片和媒体链接
- 本地存储和管理这些数据，便于用户查看和备份
- 提供推文统计功能，包括点赞数、转发数、评论数等
- 允许用户通过时间、内容类型等条件筛选数据

这是一个数据管理工具，旨在帮助用户更好地组织和管理自己的社交媒体内容，仅用于个人使用和数据分析目的。

## 需使用远程代码的理由

本扩展程序**不使用远程代码**。所有功能均在本地浏览器环境中执行：

- 所有 JavaScript 代码随扩展程序安装包一起提供
- 所有数据处理和存储均在用户本地浏览器中完成
- 不从任何远程服务器下载或执行代码
- 所有功能逻辑包含在已提交的扩展程序文件中

## 需使用主机权限的理由

本扩展程序需要以下主机权限：

1. `https://x.com/*` 和 `https://twitter.com/*`
   - 理由：扩展程序需要访问 Twitter/X 网站来提取用户自己的推文数据
   - 用途：解析页面 DOM 元素，提取推文 ID、内容、图片链接和统计信息
   - 必要性：这是实现数据提取功能的基础要求，无法通过其他方式实现

2. `https://*.vxtwitter.com/*` 和 `https://*.fixvx.com/*`
   - 理由：这些是第三方服务，用于提供高质量的视频/图片预览功能
   - 用途：获取推文媒体内容的高质量预览链接
   - 必要性：提升用户体验，但这些服务是可选的，扩展程序可在没有这些权限的情况下运行

## 需使用activeTab权限的理由

activeTab 权限用于：

1. **侧边栏功能实现**：
   - 理由：扩展程序需要在当前活动标签页中注入侧边栏界面
   - 用途：在 Twitter/X 页面上显示数据管理界面，使用户能够查看和操作抓取的数据
   - 必要性：这是扩展程序核心用户界面功能的基础

2. **实时数据更新**：
   - 理由：需要检测当前页面的变化并更新显示的数据
   - 用途：当用户滚动页面或导航到新页面时，自动更新显示的数据
   - 必要性：确保用户界面中的数据与实际页面内容保持同步

## 需使用scripting权限的理由

scripting 权限用于：

1. **内容脚本注入**：
   - 理由：扩展程序需要向 Twitter/X 页面注入 JavaScript 代码来提取数据
   - 用途：解析页面 DOM 结构，识别和提取推文相关元素
   - 必要性：这是实现数据抓取功能的技术基础

2. **侧边栏界面注入**：
   - 理由：需要在页面上创建并显示侧边栏界面
   - 用途：提供用户交互界面，显示和管理抓取的数据
   - 必要性：实现扩展程序的用户界面功能

## 需使用storage权限的理由

storage 权限用于：

1. **本地数据存储**：
   - 理由：扩展程序需要存储用户抓取的推文数据
   - 用途：保存推文内容、图片链接、统计信息等数据，供用户随时查看
   - 必要性：这是扩展程序核心功能，确保数据在浏览器会话之间持久存在

2. **用户设置保存**：
   - 理由：存储用户的界面偏好和设置
   - 用途：保存用户的筛选条件、界面布局等偏好设置
   - 必要性：提升用户体验，保持个性化的使用环境

## 数据使用确认

我确认本扩展程序的数据使用情况符合 Google Chrome Web Store 的开发者计划政策：

1. **数据最小化原则**：只收集和存储用户明确请求的推文数据，不收集任何个人信息或浏览历史
2. **本地处理**：所有数据处理和存储均在用户本地浏览器中完成，不向任何远程服务器传输数据
3. **透明性**：通过清晰的界面和文档向用户说明数据收集和使用方式
4. **用户控制**：用户完全控制数据的收集、存储和删除，可随时清除所有本地存储的数据
5. **合法合规**：扩展程序仅用于帮助用户管理自己的社交媒体内容，遵守相关平台的使用条款

本扩展程序尊重用户隐私，不进行任何形式的数据跟踪或分析，不与第三方共享用户数据。
